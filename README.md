# classroom-behavior-detection

This repository demonstrates a classroom behavior detection task using computer vision.

All steps for **dataset downloading, training, and evaluation** are provided in a single notebook.

## Getting Started

Please open and run the following notebook:

- `demo.ipynb`

The notebook includes:
- Dataset downloading and preparation
- Model training
- Model evaluation and visualization

## Notes
- The notebook is self-contained.
- No additional scripts are required.
- Recommended environment: Kaggle Notebook or Jupyter Notebook with Internet enabled.
